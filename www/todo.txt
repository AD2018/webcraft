+ неограниченный мир с постепенной подгрузкой

+ масштабируемый интерфейс (чтобы на разных разрешениях элементы интерфейса смотрелись одинаково)
+ скины игроков
+ 2-я версия скинов (2-х слойные)
+ HD текстуры (с возможностью переключения на обычные)
+ поддержка рецептов в JSON
+ физика сыпучих блоков
+ сетевая игра (сервер на GO)
+ список сетевых игроков в окне отладки (F3)
+ возможность создания собственных серверов
+ skybox небо
+ видимость сетевых игроков
+ крафтовый стол
+ возможость крафта
+ создание и логика сундука
+ сундук нельзя разрушить, если он не пустой
+ чат

+ позволять скрывать тех. информацию и интерфейс по кнопкам F3, F1
